# Sistema de Gestão de Património (Site Oficial)

Este repositório contém o código-fonte da landing page oficial do **Sistema de Gestão de Património**, uma solução corporativa desenvolvida para o controlo total de ativos, inventário e estrutura organizacional.

## 🚀 O Produto

O sistema é uma plataforma **Enterprise Asset Management (EAM)** focada em alta performance, segurança de dados e soberania tecnológica para empresas que optam por infraestrutura **On-Premise**.

### Módulos Principais:
* **Controlo de Ativos:** Gestão do ciclo de vida completo de bens.
* **Movimentações:** Rastreio imutável de transferências e atribuições.
* **Identidade & ACL:** Controlo de acessos baseado em perfis (RBAC).
* **Auditoria:** Logs detalhados para conformidade e transparência.

## 🛠️ Stack Tecnológica (Site)

O site foi construído focado em velocidade e SEO:
* **Jekyll:** Gerador de sites estáticos.
* **Tailwind CSS / Custom CSS:** Para uma interface moderna e dark-themed.
* **GitHub Pages:** Alojamento e deploy contínuo.
